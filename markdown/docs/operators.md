# Operators

You can transform different values using operators like `+` and `-`.

```quark
i32 sum = 1 + 2; // 3
\
i32 negative_number = -4;
```

You can compare different values using `==` to check for equality, or `>` and `<` for greater than and less than.

```quark
if(3 > 1)  // true
if(5 == 6) // false
```

You can do the opposite of an operation using `!` or `!=` for inequality

```quark
if(3 != 4)    // true
if(!(1 == 1)) // false
```
